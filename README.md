# autosys-docker 自动化平台

## 使用方法

### 拉取源码
```bash
git clone --depth=1 https://github.com/danxiaonuo/autosys-docker.git
```

### 使用postgres版本
```bash
cd autosys-docker-postgres
docker-compose pull
docker-compose up -d
```

### 使用mysql版本
```bash
cd autosys-docker-mysql
docker-compose pull
docker-compose up -d
```
